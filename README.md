# ZALYMPIX — The Grammys of Cannabis

Concept redesign of the ZALYMPIX cannabis competition & awards site.

**Live:** https://zalympix.vercel.app

## About
Single self-contained `index.html` (inline CSS/JS/SVG, no build step, no external assets).
Chrome / matte-black / gold award-show identity drawn from the ZALYMPIX logo.

### Sections
- Hero with live countdown to the next championship
- 2026 season pipeline (CA / OK / MI / 2027 prelims)
- What is Zalympix + blind-judging format timeline
- Award categories
- 2026 California Championship feature (lineup, activations)
- Hall of Champions — real Best-Overall podiums 2021–2025 (CA / MI / East Coast)
- Sponsor wall + emblem brand-marks
- ZNN / press, prelim signup, footer

## Deploy
Static — deploy `index.html` anywhere (Vercel, Netlify, GitHub Pages, S3).

_Concept/redesign; event details compiled from public sources. Brand emblems are original stylized monograms, not the brands' trademarked logos._
